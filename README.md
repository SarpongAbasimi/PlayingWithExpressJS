<h1 align='center'>
Building Restful API with Node & Express.
</h1>

<h4 align='center'>
To Use application.
</h4>
- ``Fork`` and ``clone this repo``.
- In your terminal cd into ``Express``.
- To run ``feature test`` type ``npm test``.
- To run application type ``npm dev run ``.
